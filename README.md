## Current Projects

### 1. Laptop Assembly & Disassembly
- Open laptops to explore internal components (RAM, SSD/HDD, battery, cooling system, motherboard)
- Practice safe removal and installation of components
- Document step-by-step process with photos and notes

### 2. Mobile Phone Teardown
- Carefully disassemble phones to identify components
- Learn how batteries, cameras, screens, and motherboards are connected
- Capture images and notes for learning and reference

### 3. Hardware Upgrades
- Upgrade RAM and SSD on laptops
- Replace old HDDs with SSDs for speed improvements
- Document before & after performance observations

### 4. Virtual Labs (Optional)
- Simulate hardware changes in virtual environments (VMware, VirtualBox)
- Test software and OS installation processes
- Document errors and troubleshooting steps
